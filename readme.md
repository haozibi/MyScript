##安装tmux

*tmux一键安装脚本*

适用于：

* centos x86
* centos x86_64

使用方法：

```
wget https://raw.githubusercontent.com/haozibi/MyScript/master/tmux.centos.all.sh
chmod a+x tmux.centos.all.sh 
./tmux.centos.all.sh
```
